输入
	1、.matrix 文件
	2、.mm 文件（模型Z轴与OpenGL Z轴重合时的MV矩阵）
输出
	.3df文件
	.log文件
	
.matrix文件格式：
%文件名
%MV矩阵
%文件名
%MV矩阵
...

.mm文件格式：
MV矩阵

.3df文件(与.matrix文件相对应)
%文件名
%3d特征
...

。。。。。。。。。。。。。。。。。。。。。
该项目中每次只处理一个模型，及其对应的图像
可以强行关闭
每次计算完一张图像的3D特征之后
都将其序列写到log文件中去
下次接着从这个序列继续处理
